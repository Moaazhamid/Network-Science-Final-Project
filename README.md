# Global Diffusion of Educational Innovations

This repository contains the code and data for a network analysis of the HundrED 2024 global innovation dataset.

## Project Overview
This project constructs a directed, weighted country-level diffusion network based on innovation origin and implementation locations. The analysis identifies key countries that function as hubs and bridges in global innovation diffusion.

## Files

- `analysis.Rmd` — main R Markdown file containing code and analysis
- `analysis.html` — rendered output with results and figures
- `data/` — dataset constructed from publicly available HundrED sources

## Reproducibility

To reproduce the analysis:
1. Open `analysis.Rmd` in RStudio
2. Run all code chunks

## Data Source

Data is compiled from publicly available HundrED 2024 Global Innovation Collection materials and related sources.
